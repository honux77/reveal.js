If you use physical server...

  • takes 2 weeks to setup
  • expensive initial cost
  • need to predict maximum utilization of server
  • hard to terminate when no more needed

Amazon Elastic Compute Cloud

  • Virtual Servers in the cloud
  • Provides simple web interfaces and APIs to control
  • Reduces time required to obtain
  • Quickly scaling up and down

Amazon Relational Database

  • support MySQL, Oracle, MS SQL, PostgreSQL
  • enhance availability and reliablity via replication
  • provides Multi-AZ for availability and durability
